SPR Enables Secure Programmable Routers

Linux and the open source community provide incredible versatility for home networking. SPR unleashes those abilities with an enhanced focus on network hardening. With this software, hobbyists and developers alike can innovate, build, and deploy flexible networks with best of class network security.

  • Learn more

Hardened 802.11 Authentication and Access Control

Give each device a unique WPA2/3 password, to create trusted device identities without certificate enrollment. This zero-install solution works for anything from a Smart toaster-oven to a next-gen laptop. Place devices into predefined or custom network groups to set their internet access, approved network peers and allowed protocols.

  • Learn more

First-Class API

Functionality and features are available through an API. Programatically define how the network connects and retreive data about the devices on the network.

  • Learn more

Made For Innovators

Built with code from the open source community, for the open source community. SPR is fully open source and homelab ready. It uses containerized services to allow for rapid prototyping, simple deployment of new features, and hot swapping of core network services, such as DNS and DHCP.
